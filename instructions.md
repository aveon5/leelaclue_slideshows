# LeelaClue TikTok/Carousel "Story Factory" Instructions

This project is an **end-to-end automated story factory**. It is designed for an AI Agent (Creative Director) to conceive scenarios and a Python toolchain (Production Crew) to render the final 1080x1920 slideshow assets.

## 0. The Creative Engine (Agent Role)

The AI Agent acts as the **Creative Director and Content Engineer**. The goal is to generate German-language TikTok/Carousel scenarios (Photo Mode) that feel "Slow Tech"—calm, wise, and introspective, based on the ancient Vedic Game of Leela.

### The Master Prompt
To start a new batch of stories, use this prompt:

> "The project here is a companion of the LeelaClue mobile app. You will find the descriptions of the cards specifically under `C:\GitHub\AppExperiment1\assets\cards_de.json`. You can ask questions to get missing information.
>
> **Role:** You are the Creative Director and Content Engineer for 'LeelaClue'.
> **Objective:** Generate [X] German-language TikTok/Carousel scenarios (Photo Mode).
> **The Vibe:** 'Slow Tech'—spiritual, deep, introspective, Vedic wisdom (Not Tarot).
> **Format:** A 7-slide carousel using a 3-Card Spread (Status/Standort, Obstacle/Hindernis, Resource/Ressource).
>
> **Your Task:** Generate a table for [X] scenarios with these columns:
> 1. **ID:** 1-[X].
> 2. **Slide 1 (The Hook):** A relatable, emotionally charged question a friend/relative once asked (German).
> 3. **Slide 2 (The In-App Question):** The specific question the user types into LeelaClue to solve the Hook (German).
> 4. **Card Assignments:** Randomly select 3 IDs from the 72 cards in `cards_de.json`.
> 5. **Card Text Overlays (Slides 3, 4, 5):**
>    - **Card 1 (Standort):** How this card reflects the current state.
>    - **Card 2 (Hindernis):** The hidden block the card reveals.
>    - **Card 3 (Ressource):** The spiritual 'clue' or solution.
> 6. **Slide 6 (The Shift):** A brief concluding thought on how the user feels now (German).
> 7. **Nano Banana Image Prompts:** Two prompts (Slide 1 and Slide 6) for atmospheric, moody, spiritual background images.
>
> **Constraint:** No fortune-telling; focus on mirroring the present moment. All text must be German."

### Follow-up Work
Once the agent understands the context, the user can simply say:
**"Generate the next 14 stories (or 20)."**

## 1. Project Structure

- `tiktok_scenarios.json`: The source of truth (Generated by Agent).
- `tiktok_nano_banana_prompts.json`: Image generation prompts (Generated by Agent).
- `scenario_assets/`: Storage for generated images, by `scenario_XX`.
- `empty.jpg`: Template for **Slide 2** (The Question).
- `emblem2.png`: Source for **Slide 7** (The Final CTA).

---

## 2. Production Workflow (Production Crew)

After the Agent provides the JSON data, execute these steps:

### Phase 1: Background Generation
- **Manual:** Upload manually generated hook and shift high-quality images and place them as `1_Hook.jpg` and `6_Shift.jpg`.

### Phase 2: Text Rendering
1. **Overlay Text (Slides 1, 3, 4, 5, 6):** `python overlay_text.py --force`
   - Applies dual-color Hook text, card titles at the top, and professional shadows.
2. **Question Slide (Slide 2):** `python generate_question_slides.py --force`
   - Auto-fits text into the `empty.jpg` card template using Philosopher font.
3. **Finish Slide (Slide 7):** `python generate_emblem_slide.py`
   - Renders the final branding and "Probiere die App..." CTA.

### Phase 3: Cloud Upload (Optional)
To automatically push the rendered `_Text.jpg` assets to a remote Google Drive folder, run:
`python upload_to_drive.py --drive_root_folder_id <YOUR_FOLDER_ID>`

**Prerequisites for Uploading:**
You will need a `credentials.json` file in the project root. To get this:
1. Go to the [Google Cloud Console](https://console.cloud.google.com/).
2. Create a new Project (or select an existing one).
3. Go to **APIs & Services > Library** and search for **Google Drive API**. Click **Enable**.
4. Go to **APIs & Services > OAuth consent screen**. Select "External" and fill in the required fields (App name, support email, developer email). Under scopes, you don't *strictly* need to add anything here right now, but you can add the `.../auth/drive.file` scope. Save and continue. Add your own Google Account email under "Test users".
5. Go to **APIs & Services > Credentials**. Click **Create Credentials > OAuth client ID**.
6. Select **Desktop App** as the Application type and give it a name to create it.
7. Click the Download icon next to the newly created credential.
8. Rename the downloaded file to exactly `credentials.json` and place it in the same directory as the script.

## 3. 🎨 Visual Safety Zone Guide

When generating or reviewing assets, remember TikTok's UI requirements for 1080x1920:

- **Avoid the Right Edge:** Like, Comment, and Share buttons live here.
- **Avoid the Bottom 20%:** Caption and "Photo Mode" indicators sit here.
- **Avoid the Top 10%:** "Following/For You" tabs are located here.
- **The "Golden Center":** Keep card art and text overlays in the middle 60% for maximum readability.
